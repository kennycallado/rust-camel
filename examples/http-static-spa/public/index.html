HTTP Static SPA Example

SPA Fallback Active

This page is served by http-static with SPA fallback enabled. Any unmatched client-side route (e.g., /login, /settings) falls back to this index.html.

Routing Demo

  • /api/hello — API route (handled by http:)
  • /dashboard — SPA fallback → index.html (no file at path)
  • /nonexistent — SPA fallback → index.html
  • /missing.css — custom 404 page (file extension blocks SPA fallback, errorPages.404 kicks in)
